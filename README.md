# GoogleKeySearch

Google 検索をキーボードショートカットで快適に操作できる Chrome 拡張機能です。

## 機能

- **検索結果ナビゲーション** — `j` / `k` で上下に移動し、`Enter` で開く
- **検索候補ナビゲーション** — サジェストドロップダウンが開いているとき `j` / `k` で選択
- **タブ切り替え** — ワンキーで各カテゴリタブに移動
- **ページ送り** — `l` で次のページ、`h` で前のページ
- **キーのカスタマイズ** — ポップアップから全ショートカットを変更可能
- **キーマップバー** — 現在の割り当てを検索ページ上部に常時表示

## デフォルトキー

| キー | 操作 |
|------|------|
| `j`  | 検索候補・結果を下に移動 |
| `k`  | 検索候補・結果を上に移動 |
| `a`  | すべてタブ |
| `i`  | 画像タブ |
| `v`  | 動画タブ |
| `m`  | 地図タブ |
| `s`  | ショッピングタブ |
| `n`  | ニュースタブ |
| `l`  | 次のページ |
| `h`  | 前のページ |

> 入力フィールドにフォーカスがある場合、タブ切り替え・ページ送りショートカットは無効になります。  
> 検索候補ナビゲーション（j/k）はドロップダウンが開いているときのみ有効です。

## インストール

Chrome Web Store には未公開のため、デベロッパーモードで手動インストールします。

1. このリポジトリをクローン（または ZIP でダウンロード・展開）
   ```bash
   git clone https://github.com/<your-username>/googlekeysearch.git
   ```
2. Chrome のアドレスバーに `chrome://extensions` と入力して開く
3. 右上の **デベロッパーモード** をオンにする
4. **パッケージ化されていない拡張機能を読み込む** をクリック
5. クローンしたフォルダを選択

## キーのカスタマイズ

Chrome ツールバーの拡張機能アイコンをクリックすると設定ポップアップが開きます。  
変更したいキーをクリックし、新しいキーを押して **保存** してください。  
**デフォルトに戻す** ボタンで初期値に戻せます。

## ファイル構成

```
googlekeysearch/
├── manifest.json   # 拡張機能の定義（Manifest V3）
├── content.js      # Google 検索ページに注入するスクリプト
├── popup.html      # 設定ポップアップの UI
└── popup.js        # 設定ポップアップのロジック
```
